# 🔐 Caesar Cipher - Encryption & Decryption

## DecodeLabs Cybersecurity Project 2 | Batch 2026

### 📌 Overview
Classic Caesar cipher implementation that shifts letters by a user-defined key. Demonstrates symmetric encryption concepts.

### ✨ Features
- Encrypt any text with custom shift (1-25)
- Decrypt ciphertext back to original
- Preserves case and non-letters
- Brute-force attack demo
- Frequency analysis visualization
- Interactive mode

### 🚀 How to Run
```bash
python caesar_cipher.py
